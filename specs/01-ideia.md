# Ideia do Projeto — AgroMassa

## Nome do projeto
AgroMassa

## Tipo de projeto
Site institucional e comercial para divulgação e consulta de tratores e implementos agrícolas.

## Objetivo do projeto
Criar um site para a AgroMassa que apresente a empresa, mostre os produtos disponíveis e facilite o contato de clientes interessados por meio do WhatsApp.

O site não realizará compras online. O objetivo é permitir que o visitante visualize os produtos, consulte informações e entre em contato diretamente com a AgroMassa para negociar.

## Público-alvo
- Produtores rurais
- Fazendeiros
- Empresas do agronegócio
- Compradores de tratores
- Compradores de implementos agrícolas
- Pessoas interessadas em compra, venda, troca ou locação de máquinas agrícolas

## Estrutura principal do site
O site deve ter pelo menos estas áreas principais:

### 1. Página institucional da AgroMassa
Uma página para apresentar a empresa e suas informações principais.

Deve conter:
- Nome da empresa
- Logo da AgroMassa
- Descrição da empresa
- Informações de contato
- Número de WhatsApp
- Cidade e estado
- Serviços oferecidos, como compra, venda, troca e locação
- Seção sobre a empresa
- Rodapé com contatos e links úteis

### 2. Página pública de produtos
Uma página onde o visitante pode visualizar os produtos disponíveis.

Deve conter:
- Listagem de tratores e implementos agrícolas
- Busca por texto
- Filtros de produtos
- Cards ou lista de produtos
- Página ou modal de detalhes do produto
- Botão para chamar no WhatsApp em cada produto
- Galeria de fotos do produto

O visitante não poderá comprar pelo site. Ele apenas:
- visualiza os produtos
- consulta os detalhes
- entra em contato via WhatsApp quando tiver interesse

### 3. Área administrativa
Uma área restrita para administração do conteúdo.

Deve permitir:
- adicionar produtos
- editar produtos
- remover produtos
- cadastrar fotos dos produtos
- editar informações da empresa
- controlar quais produtos aparecem no site
- marcar produtos como destaque, disponível, vendido ou sob consulta

## Categorias principais de produtos
O site deve trabalhar com duas categorias principais:
- Tratores
- Implementos agrícolas

## Exemplos de implementos agrícolas
- Grades
- Arados
- Plantadeiras
- Pulverizadores
- Carretas agrícolas
- Roçadeiras
- Subsoladores
- Distribuidores
- Outros implementos

## Funcionalidades principais do MVP

### Área pública
- Exibir página institucional da AgroMassa
- Exibir catálogo de produtos
- Mostrar detalhes do produto
- Mostrar fotos do produto
- Permitir busca por nome, marca, modelo ou descrição
- Permitir filtro por categoria
- Permitir filtro por condição do produto
- Redirecionar o usuário para o WhatsApp ao demonstrar interesse

### Área administrativa
- Login do administrador
- Cadastro de produtos
- Edição de produtos
- Exclusão de produtos
- Upload de fotos dos produtos
- Edição das informações institucionais da AgroMassa

## Campos necessários para cadastro de produto
Cada produto deve ter os seguintes campos:

- ID
- Nome do produto
- Categoria
- Tipo/subcategoria
- Marca
- Modelo
- Ano
- Condição do produto (novo, usado, seminovo)
- Descrição
- Especificações técnicas
- Preço (opcional)
- Status do produto (disponível, vendido, sob consulta, alugado, etc.)
- Cidade/estado
- Destaque (sim ou não)
- Foto principal
- Fotos adicionais / galeria
- Link ou botão de contato via WhatsApp

## Regras importantes do projeto
- O site não terá checkout
- O site não terá carrinho de compras
- O site não processará pagamento online
- O interesse do cliente será tratado pelo WhatsApp
- Todo produto deve ter botão de contato via WhatsApp
- Todo produto deve permitir exibição de fotos
- A área pública é apenas para visualização e consulta
- A área administrativa é exclusiva para gestão do conteúdo

## Busca e filtros
A página de produtos deve permitir:
- busca por palavra-chave
- filtro por categoria
- filtro por condição do produto
- filtro por marca, se possível no MVP
- filtro por status, se possível no MVP

## Identidade visual desejada
A identidade visual deve seguir a logo da AgroMassa anexada neste projeto.

Características desejadas:
- estilo forte e profissional
- visual ligado ao agronegócio
- uso predominante de verde, preto, branco e tons neutros
- aparência confiável, robusta e moderna
- layout responsivo, com boa experiência em celular

## Referência visual
A logo da AgroMassa está anexada ao projeto e deve ser usada como referência de:
- cores
- estilo visual
- personalidade da marca

## Objetivo do MVP
Ter um site funcional com:
- página institucional
- página pública de produtos
- busca e filtros
- visualização detalhada dos produtos
- redirecionamento para WhatsApp
- área administrativa para cadastrar e editar produtos

## Fora do escopo inicial
- pagamento online
- carrinho de compras
- cálculo de frete
- marketplace multiusuário
- chat interno no site
- múltiplos perfis administrativos complexos
- integração com ERP
- integração com estoque avançado
